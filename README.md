# MACSO

Author: Bryan Cisneros  
Date: January 2024

This repository contains small experiment scripts (PCA, K-Means, Fuzzy C-Means, KNN, SVM, Random Forest) used to explore machine-learning techniques on the provided datasets.

## Data
- `conjunto_de_datos_normalizados.xlsx`: normalized dataset (features + target label)
- `alumnos.csv`: raw dataset snapshot

## Notes
- Files were standardized to remove external web references and to use consistent English headers.
- Each `*.py` file is intended to be run as a standalone script.
